Text Editor Bar

TextEditorBar is jQuery Plugin that allow you to create custom textarea content.


Its higly customizable:

  1. You can choose which button to show or not
  2. You can also change the button designs as you want
  3. You can also add the proprties like add more colors or smiles
  4. You can choose if should be converted to HTML entities
